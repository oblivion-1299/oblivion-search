# Contributing to OBLIVION Search

Thank you for your interest in OBLIVION Search!

## How to Contribute

### Bug Reports
- Open an [Issue](https://github.com/oblivion-1299/oblivion-search/issues)
- Include: steps to reproduce, expected behavior, actual behavior
- Screenshots or error messages help a lot

### Security Issues
- Do NOT open a public issue
- Email admin@oblivionzone.com instead
- See [SECURITY.md](SECURITY.md)

### Feature Requests
- Open an Issue with the "enhancement" label
- Describe the feature and why it would be useful

### Code Contributions
This is a source-available project. While we welcome bug reports and suggestions, code contributions require a Contributor License Agreement (CLA) due to the license terms.

Contact admin@oblivionzone.com to discuss code contributions.

### Privacy Policy Review
We especially welcome feedback on our [Privacy Policy](https://oblivionsearch.com/privacy). If you spot gaps or concerns, please open an issue.

## Code of Conduct

Be respectful. Be constructive. Focus on the code, not the person.
